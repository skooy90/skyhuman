Mat_list.jsp 구현 계획
구현할 기능:
1. 필터링 기능
검색: 재고코드, 제품명으로 검색
제품유형별 필터 버튼:
"전체" (기본)
"원자재"
"반제품"
"완제품"
버튼 클릭시 해당 유형만 표시
2. 정렬 기능
재고량 기준 정렬: 많은 순, 적은 순 (드롭다운)
3. 상태 표시
재고 부족 알림: 재고량이 일정 수준 이하일 때 경고 표시
재고 상태: 정상/부족/과다 (색상으로 구분)
4. 페이지네이션
한 페이지당 10개씩 표시
페이지 번호 네비게이션
테이블 컬럼:
재고코드 (materialCode)
제품코드 (standardCode)
제품명 (stName)
제품유형 (stType)
재고량 (maQuantity) + 상태 표시
단위 (stUnit)
담당자 (employeeNo)
생성일 (maCreationDate)
수정일 (maUpdateDate)
관리 (수정/삭제)
