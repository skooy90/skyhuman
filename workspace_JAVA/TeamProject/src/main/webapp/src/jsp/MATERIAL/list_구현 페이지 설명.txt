# 📋 재고 관리 페이지 구현 설명

## 🎯 구현된 페이지

### 1. Mat_list.jsp (재고 목록 페이지)
**파일 위치**: `/src/jsp/MATERIAL/Mat_list.jsp`

#### 📊 주요 기능
- **재고 목록 조회**: 원자재, 반제품, 완제품 전체 표시
- **필터링 기능**: 
  - 검색: 재고코드, 제품명으로 검색
  - 제품유형별 필터: 전체/원자재/반제품/완제품 버튼
- **정렬 기능**: 재고량 많은 순/적은 순, 재고코드 순, 제품명 순
- **상태 표시**: 정상(초록), 부족(빨강), 과다(노랑) 배지
- **페이지네이션**: 한 페이지당 10개씩 표시
- **통계 정보**: 전체 재고 수, 재고 부족 수, 정상 재고 수

#### 🧪 오이비누 제조업 샘플 데이터
**원자재 (6종)**:
- 신선 오이 (25kg) - 재고 부족 ⚠️
- 가성소다 (150g) - 정상
- 팜 오일 (200g) - 정상
- 올리브 오일 (130g) - 정상
- 살리실산 (15g) - 재고 부족 ⚠️
- 아연 PCA (80g) - 정상

**반제품 (4종)**:
- 오이 비누 베이스 (500kg) - 정상
- 오이 추출물 (30L) - 정상
- 각질제거 성분 (25g) - 정상
- 수축 성분 (20g) - 정상

**완제품 (3종)**:
- 오이 천연비누 (1,200개) - 재고 과다 📈
- 오이 각질제거비누 (580개) - 정상
- 오이 수축비누 (780개) - 정상

#### 🎨 UI 특징
- **반응형 디자인**: 모바일/태블릿 대응
- **일관된 스타일**: 기존 프로젝트와 동일한 색상 테마
- **직관적 인터페이스**: 필터 버튼, 검색창, 정렬 드롭다운
- **상태 시각화**: 색상과 배지로 재고 상태 구분

#### 🔗 연결된 페이지
- **재고 등록**: `Mat_form.jsp` (아직 미구현)
- **수정 링크**: `Mat_form.jsp?code=MA0001` ~ `Mat_form.jsp?code=MA0013`

## 🎯 MaterialDTO 기반 구현
**사용된 필드**:
- `materialCode`: 재고코드 (MA0001, MA0002...)
- `standardCode`: 제품코드 (RA001, SE001, FI001...)
- `maQuantity`: 재고량 (25, 150, 200...)
- `employeeNo`: 담당자 (김재고, 이공정, 박품질)
- `maCreationDate`: 생성일
- `maUpdateDate`: 수정일

**추가 표시 정보** (StandardDTO와 조인 예정):
- `stName`: 제품명 (신선 오이, 오이 비누 베이스...)
- `stType`: 제품유형 (원자재, 반제품, 완제품)
- `stUnit`: 단위 (kg, g, L, 개)
