Process_detail.jsp 페이지 설명
1. 페이지 개요
파일명: Process_detail.jsp
목적: 특정 제품의 공정 상세 정보 및 작업 지침서 제공
URL: Process_detail.jsp?code=FI001
기능: 제품별 공정 순서, 작업 방법, 필요 자재 정보 표시
2. 핵심 기능 - 작업 지침서
이 페이지의 가장 중요한 기능은 실제 작업자가 공정을 수행할 수 있는 상세한 작업 지침서를 제공하는 것입니다.
작업 방법 섹션:
단계별 가이드: 1, 2, 3... 순서로 작업 방법 설명
구체적인 수량: kg, ml, g 단위로 정확한 수량 표시
정확한 온도: 40°C, 35°C 등 온도 정보
정확한 시간: 3분, 30분, 24시간, 4주 등 시간 정보
필요 자재 섹션:
자재명: 신선 오이, 가성소다, 팜 오일 등
정확한 수량: 2kg, 200g, 1kg 등
장비: 믹서기, 교반기, 성형틀, 건조실 등
3. 제품별 작업 지침 예시
오이 비누 베이스 (SE001 - 반제품):
1단계: 원료 혼합 - 오이 2kg 씻기 → 슬라이스 → 믹서기로 갈기
2단계: 비누 베이스 제조 - 가성소다 200g 용해 → 팜 오일 1kg 가열 → 혼합
3단계: 건조 - 성형 틀에 부어넣기 → 24시간 경화 → 4주간 건조
4단계: 베이스 포장 - 개별 포장 → 베이스 라벨 부착 → 박스 포장
오이 천연비누 (FI001 - 완제품):
5단계: 성분 추가 - 오이 1kg 씻기 → 믹서기로 갈기 → 추출물 분리
6단계: 비누 베이스 혼합 - 베이스 2kg 가열 → 올리브 오일 100ml 추가
6단계: 성분 배합 - 비타민 E 오일 10ml → 천연 보습제 50ml → 에센셜 오일 5ml
7단계: 성형 및 건조 - 성형 틀에 부어넣기 → 12시간 경화 → 2주간 건조
8단계: 포장 - 개별 포장 → 라벨 부착 → 품질 검사
4. UI/UX 특징
시각적 구분: 작업 방법(파란색), 필요 자재(노란색)
단계별 번호: 파란색 원형 배지로 순서 표시
반응형 디자인: 데스크톱(2열), 모바일(1열)
직관적 네비게이션: 명확한 단계별 구분
5. 핵심 가치
작업자 중심: 실제 작업 가능한 구체적 정보
품질 관리: 모든 작업자가 동일한 방법으로 작업
효율성: 작업 방법을 빠르게 확인하여 시간 절약
교육 효과: 신입 작업자 교육 자료로 활용
이 페이지는 단순한 정보 표시가 아니라 실제 제조 현장에서 사용할 수 있는 작업 지침서 역할을 하는 것이 핵심입니다!
