BOM_detail.jsp 구현 계획

1. 페이지 개요
- BOM 상세 정보 조회 페이지
- URL 파라미터로 제품코드 받음 (?code=FI001)
- BOM 정보, 자재 목록, 수량 계산기, 재고 현황 제공
- 작업 지침서와 연계된 자재 소요량 계산 도구

2. 페이지 구조
2.1 헤더 섹션
- 제품명 + 제품 유형 + BOM 설명
- 액션 버튼: 수정, 삭제, 목록으로
- BOM 기본 정보 (BOM번호, BOM 순서, BOM 유형)

2.2 기본 정보 카드
- 제품 정보 (StandardDTO 기반)
- BOM 정보 (BOMDTO 기반)
- 등록일, 수정일, 등록자 정보

2.3 BOM 자재 표
- 해당 제품의 모든 자재 정보
- 자재별 상세 정보 및 현재 재고 현황
- 재고 상태 표시 (충분/부족/없음)

2.4 수량 계산기
- 생산 수량 입력 필드
- 자동 계산: BOM 수량 × 생산 수량 = 총 필요 수량
- 실시간 재고 비교 및 부족 재고 알림

2.5 재고 현황 요약
- 전체 자재 현황 통계
- 부족 재고 목록
- 재고 충족률 표시

3. BOM 자재 표 구조
3.1 표 헤더
- 자재코드, 자재명, 자재 유형, 단위
- BOM 수량, 현재 재고, 재고 상태, 비고

3.2 자재 정보
- 자재코드: MA0001, MA0002, MA0003...
- 자재명: 신선 오이, 천연 비누 베이스, 오이 추출물...
- 자재 유형: 원자재, 반제품, 완제품
- 단위: kg, ml, g, 개
- BOM 수량: 1개 제품당 필요 수량
- 현재 재고: MaterialDTO에서 조회한 실제 재고
- 재고 상태: 충분(초록)/부족(노랑)/없음(빨강)
- 비고: 추가 설명

4. 수량 계산기 기능
4.1 입력 섹션
- 생산 수량 입력 필드 (숫자)
- 계산 버튼
- 초기화 버튼

4.2 계산 결과 표
- 자재별 필요 수량 (BOM 수량 × 생산 수량)
- 현재 재고와 비교
- 부족 수량 표시
- 재고 충족 여부 표시

4.3 JavaScript 기능
- 실시간 계산 (입력 즉시 결과 표시)
- 재고 부족 시 경고 표시
- 계산 결과 하이라이트

5. 재고 현황 요약
5.1 통계 카드
- 전체 자재 수
- 충분한 재고 자재 수
- 부족한 재고 자재 수
- 재고 충족률 (%)

5.2 부족 재고 목록
- 부족한 자재 목록
- 부족 수량 표시
- 구매/발주 권장 메시지

7. UI/UX 기능
7.1 반응형 디자인
- 모바일 대응
- 테이블 가로 스크롤
- 카드 레이아웃

7.2 인터랙티브 요소
- 수량 입력시 실시간 계산
- 재고 상태별 색상 구분
- 부족 재고 하이라이트
- 툴팁으로 상세 정보 표시

7.3 액션 버튼
- 수정: BOM_form.jsp로 이동
- 삭제: 확인 후 삭제
- 목록으로: BOM_list.jsp로 이동
- 인쇄: BOM 정보 인쇄

8. JavaScript 기능
8.1 수량 계산기
- 입력값 변경시 실시간 계산
- 재고 부족시 경고 표시
- 계산 결과 테이블 업데이트

8.2 재고 상태 표시
- 재고량에 따른 상태 색상 변경
- 부족 재고 하이라이트
- 통계 정보 실시간 업데이트

8.3 폼 검증
- 생산 수량 유효성 검사
- 숫자 범위 검증
- 에러 메시지 표시

9. 스타일링
9.1 카드 레이아웃
- 기본 정보 카드
- 수량 계산기 카드
- 재고 현황 카드
- 통계 카드

9.2 테이블 스타일
- BOM 자재 표
- 계산 결과 표
- 부족 재고 목록

9.3 상태 표시
- 재고 상태 배지
